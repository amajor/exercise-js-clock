# JS Exercise

## Date & Time

This exercise will walk you through display the current date and time on a web page using plain javascript.

You'll cover the following things:

* document / DOM
* getElementById()
* textContent
* Date()
  * getMonth()
  * getDate()
  * getFullYear()
  * getHours()
* ternary operator
* remainder operator
* functions
* setTimeout()
